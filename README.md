# Personal Portfolio

A professional, high-impact personal portfolio showcasing my expertise in AI/ML, Cybersecurity, and Cloud computing.

## 🚀 Features
- **Modern Design**: Built with glassmorphism and smooth animations.
- **Responsive**: Fully optimized for all device sizes.
- **Project Showcases**: Detailed highlights of key projects like RailRakshak and AI Spyware Detection.
- **Tech Stack**: Visual representation of my technical skills.

## 🛠️ Built With
- **HTML5**
- **Vanilla CSS** (Custom Design System)
- **JavaScript** (Animations & Interactions)
- **Google Fonts** (Inter & Outfit)
- **Font Awesome Icons**

## 📂 Project Structure
- `index.html`: Main portfolio structure.
- `style.css`: Custom styling and design tokens.
- `script.js`: Interactive elements and navigation logic.
- `assets/`: Project images and media.
